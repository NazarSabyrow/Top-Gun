Эта игра разработана по мотивам культового фильма "Топ Ган" 1987 года и представляет собой полностью браузерную 2D авиационную боевую игру. Игрок управляет 
военным самолетом, сражается с вражескими aircraft и пытается завершить уровни.
Клавиши направления (←↑↓→): Управление движением самолета
Клавиша Space (Пробел): Стрельба
Адаптивный дизайн: Подстраивается под разные размеры экранов
Непрерывность: Игра идет в постоянном потоке, без временных ограничений
Цель: Набирать определенное количество очков на каждом уровне для перехода на следующий
Риск: Избегать вражеского огня и столкновений
HTML5 Canvas: Отрисовка и визуализация
RequestAnimationFrame: Высокопроизводительный игровой цикл
CSS3 Animations: Анимации интерфейса и эффектов
